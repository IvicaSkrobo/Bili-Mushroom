## Decisions Made

- **react-leaflet v4 (not v5):** v5.0.0 requires React 19 as a peer dependency; project uses React 18. Installed v4.2.1 which supports React 18. This is the correct version — v5 is documented as requiring React 19 upgrade.
- **Full mock in tests:** jsdom cannot render Leaflet maps (no canvas/SVG DOM support). Both `react-leaflet` and `leaflet` are fully mocked in the test file. Marker PNG assets are also mocked.
- **key={String(open)} remount strategy:** Simpler than forceMount + CSS visibility; acceptable for a picker modal opened infrequently.
