## Decisions Made
- **Cross-module test helpers:** Placed `setup_in_memory_db` and `make_find_record` in `pub(crate) mod test_helpers` at the module level in import.rs (outside `mod tests`), so finds.rs tests can import them. Items inside `#[cfg(test)] mod tests` are not reachable from other modules even with `pub(crate)`.
- **`insert_find_row` and `insert_find_photo` made `pub(crate)`:** Necessary for finds.rs tests to exercise DB state setup without duplicating code.
- **Mode A additional_photos loop:** Each additional source path gets its own sequence number via `next_seq_for_folder`, is copied to the same destination folder, and inserted as `is_primary = false`.
