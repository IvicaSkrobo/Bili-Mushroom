**Structs (all derive `serde::Serialize, Clone, Debug`):**

1. `StatsCards` — `total_finds: i64`, `unique_species: i64`, `locations_visited: i64`, `most_active_month: Option<String>` (format "YYYY-MM", e.g. "2024-05"; frontend formats display)

2. `TopSpot` — `country: String`, `region: String`, `location_note: String`, `count: i64`

3. `BestMonth` — `month_num: u8` (1-12), `count: i64`

4. `CalendarEntry` — `month: u8` (1-12), `species_name: String`, `date_found: String`, `location_note: String`

5. `SpeciesStatSummary` — `species_name: String`, `find_count: i64`, `first_find: String`, `best_month: Option<String>` (format "YYYY-MM"), `locations: Vec<SpeciesLocation>`

6. `SpeciesLocation` — `country: String`, `region: String`, `location_note: String`

**Commands:**

`get_stats_cards(storage_path: String) -> Result<StatsCards, String>`:
- `SELECT COUNT(*) FROM finds` for total_finds
- `SELECT COUNT(DISTINCT species_name) FROM finds` for unique_species
- `SELECT COUNT(DISTINCT country || '|' || region || '|' || location_note) FROM finds` for locations_visited
- `SELECT strftime('%Y-%m', date_found) as ym, COUNT(*) as cnt FROM finds GROUP BY ym ORDER BY cnt DESC LIMIT 1` for most_active_month (use `.ok()` to convert to Option)

`get_top_spots(storage_path: String) -> Result<Vec<TopSpot>, String>`:
- `SELECT country, region, location_note, COUNT(*) as cnt FROM finds GROUP BY country, region, location_note ORDER BY cnt DESC LIMIT 10`
- Use `query_map` + `.collect::<Result<Vec<_>, _>>()` pattern from `finds.rs`

`get_best_months(storage_path: String) -> Result<Vec<BestMonth>, String>`:
- `SELECT CAST(strftime('%m', date_found) AS INTEGER) as month_num, COUNT(*) as cnt FROM finds GROUP BY month_num ORDER BY cnt DESC`
- Returns all 12 months (or fewer if not all months have finds)

`get_calendar(storage_path: String) -> Result<Vec<CalendarEntry>, String>`:
- `SELECT CAST(strftime('%m', date_found) AS INTEGER) as month, species_name, date_found, location_note FROM finds ORDER BY month ASC, date_found ASC`

`get_species_stats(storage_path: String) -> Result<Vec<SpeciesStatSummary>, String>`:
- First query: `SELECT species_name, COUNT(*) as find_count, MIN(date_found) as first_find FROM finds GROUP BY species_name ORDER BY find_count DESC`
- For each species, run two sub-queries:
  - Best month: `SELECT strftime('%Y-%m', date_found) as ym, COUNT(*) as cnt FROM finds WHERE species_name = ?1 GROUP BY ym ORDER BY cnt DESC LIMIT 1`
  - Locations: `SELECT DISTINCT country, region, location_note FROM finds WHERE species_name = ?1`
- Assemble into `SpeciesStatSummary` with embedded `locations: Vec<SpeciesLocation>`

`read_photos_as_base64(storage_path: String, photo_paths: Vec<String>) -> Result<Vec<String>, String>`:
- For each relative path in `photo_paths`, construct absolute: `format!("{}/{}", storage_path, rel)`
- Read bytes with `std::fs::read(&abs)`, encode with `base64::Engine::encode(&base64::engine::general_purpose::STANDARD, &bytes)`
- Import: `use base64::Engine;` (the base64 crate 0.22 is already in Cargo.toml)
