# Improve CollectionPins label visibility

## Goal

Map stays clean when zoomed out; species names appear when zoomed in close enough. Mixed-species pins at the same location show a compact grouped label instead of overlapping names.

## Scope

- `src/components/map/CollectionPins.tsx` — all logic changes
- `src/components/map/CollectionPins.test.tsx` — new test cases

No other files change.

## Tasks

### T1 — Collection interface: add labelText + suppressLabel

### T2 — collectionsFromFinds: post-processing proximity pass

After building the result array, group Collections by proximity using SAME_LOCATION_DEG:

```ts
const locationGroups: Collection[][] = [];
for (const col of result) {
  let placed = false;
  for (const group of locationGroups) {
    if (group.some(
      (other) =>
        Math.abs(other.lat - col.lat) <= SAME_LOCATION_DEG &&
        Math.abs(other.lng - col.lng) <= SAME_LOCATION_DEG,
    )) {
      group.push(col);
      placed = true;
      break;
    }
  }
  if (!placed) locationGroups.push([col]);
}

for (const group of locationGroups) {
  const speciesInGroup = new Set(group.map((c) => c.name));
  if (speciesInGroup.size === 1) {
    const latinName = group[0].name.split(',')[0].trim();
    for (const col of group) {
      col.labelText = latinName;
      col.suppressLabel = false;
    }
  } else {
    group[0].labelText = `${speciesInGroup.size} species`;
    group[0].suppressLabel = false;
    for (let i = 1; i < group.length; i++) {
      group[i].labelText = '';
      group[i].suppressLabel = true;
    }
  }
}
```
