## Changes

1. **CSP fix** (`tauri.conf.json`) — added `connect-src` directive with `nominatim.openstreetmap.org`. The `default-src` policy was blocking all fetch() calls to external domains including Nominatim, so geocoding silently returned empty strings.

3. **Pre-fill new photos** (`ImportDialog.tsx`) — `applyShared()` helper applies current shared header state to freshly-picked photos. EXIF date takes precedence; shared values fill in where EXIF is absent.

4. **Shared Notes** (`ImportDialog.tsx`) — Textarea added to shared header. Pre-fills new cards only (no live-cascade effect), independent from per-photo notes fields.

5. **EditFindDialog autocomplete** (`EditFindDialog.tsx`) — loads species folders from storagePath when dialog opens, shows chip suggestions while typing species name.
